**Bondage Studio** 是一个由社区运营的 GitHub 组织，为 [Bondage Club](https://www.bondageprojects.com/)
社区构建开源的工具、模组、客户端与各类实用程序。

## 这里有什么

- **扩展与模组**：为游戏添加新功能。
- **客户端**：无头与本地版本，用于自动化、测试与开发。
- **机器人与服务**：为社区提供支持。

## 参与贡献

这里的一切都是开源的。欢迎浏览
[GitHub 上的组织](https://github.com/bondage-studio)、提交 issue 或发起 pull request。
我们欢迎新人加入。

> 我们随时欢迎你帮忙翻译。
