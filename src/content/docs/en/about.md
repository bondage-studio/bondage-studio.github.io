**Bondage Studio** is a community-run GitHub organization that builds open-source
tools, mods, clients, and utilities for the [Bondage Club](https://www.bondageprojects.com/)
community.

## What lives here

- **Extensions & mods** that add features to the game.
- **Clients** — headless and local — for automation, testing, and development.
- **Bots & services** that support the community.

## Contributing

Everything here is open source. Browse the
[organization on GitHub](https://github.com/bondage-studio), open an issue, or
send a pull request. Newcomers are welcome.

> This site is multilingual. If a page hasn't been translated into your language
> yet, you'll see the English version — and translations are always welcome.
